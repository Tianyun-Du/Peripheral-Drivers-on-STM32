# HAL Configuration
The STM32 I used for driving WS2812 matrix screen is STM32F103C8T6.

## Clock Configuration

HCLK: 64MHz

## Pinout & Configuration

**SPI1 MODE**: 	Transmit Only Master

**Hardware NSS Signal**: 	Disable

## Parameter Settings 

**Frame Format**: Motorola

**Data Size**: 8 Bits

**First Bit**: MSB first

**Prescaler**: 16

**CPOL**: LOW

**CPHA**: 1 Edge

## DMA Settings
Add DMA channel

SPI1_TX DMA1 Channel3 Memoty to Peripheral Low priority

**DMA Request Settings** : Normal Mode, Memory Increament Address, Data width Byte
